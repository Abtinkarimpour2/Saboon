# راهنمای اضافه کردن لوگو

## مراحل:

1. فایل لوگوی خود را با نام `logo.png` در پوشه `public` قرار دهید
2. فرمت‌های پشتیبانی شده: PNG, JPG, SVG
3. اندازه پیشنهادی: 
   - عرض: 200-300px
   - ارتفاع: متناسب با عرض
   - پس‌زمینه: شفاف یا سفید

## مسیر فایل:

```
public/
  └── logo.png
```

## نکات:

- اگر لوگو اضافه نشود، متن "Biaresh" به صورت خودکار نمایش داده می‌شود
- لوگو در Header و Footer نمایش داده می‌شود
- لوگو به صفحه اصلی لینک می‌شود
